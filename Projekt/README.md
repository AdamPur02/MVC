# \# ExpenseTracker

# 

# \## Opis projektu

# 

# ExpenseTracker to prosta aplikacja webowa stworzona w technologii ASP.NET Core MVC. Jej zadaniem jest umożliwienie użytkownikowi zarządzania domowymi wydatkami.

# 

# Aplikacja pozwala na dodawanie, przeglądanie, edytowanie oraz usuwanie wydatków zapisanych w bazie danych SQLite.

# 

# Projekt został wykonany w ramach zaliczenia przedmiotu dotyczącego wzorca architektonicznego MVC.

# 

# \## Wykorzystane technologie

# 

# \* C#

# \* ASP.NET Core MVC

# \* Entity Framework Core

# \* SQLite

# \* Bootstrap

# 

# \## Funkcjonalności

# 

# Aplikacja umożliwia:

# 

# \* dodawanie nowych wydatków,

# \* wyświetlanie listy wszystkich wydatków,

# \* edycję istniejących wydatków,

# \* usuwanie wydatków,

# \* przeglądanie szczegółów wybranego wydatku,

# \* filtrowanie wydatków według kategorii,

# \* walidację danych w formularzach.

# 

# Każdy wydatek zawiera:

# 

# \* kategorię,

# \* kwotę,

# \* datę.

# 

# \## Uruchomienie projektu

# 

# 1\. Otworzyć projekt w Visual Studio.

# 2\. Odbudować rozwiązanie.

# 3\. Uruchomić aplikację.

# 4\. W razie potrzeby wykonać migracje bazy danych Entity Framework.

# 5\. Po uruchomieniu aplikacja będzie dostępna w przeglądarce internetowej.

# 

# \## Struktura MVC

# 

# \### Model

# 

# Model `Expense` przechowuje informacje o wydatkach i odpowiada za walidację danych.

# 

# \### View

# 

# Widoki odpowiadają za prezentację danych użytkownikowi oraz obsługę formularzy.

# 

# \### Controller

# 

# Kontroler `ExpensesController` obsługuje operacje CRUD oraz filtrowanie danych.
